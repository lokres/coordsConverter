<h3>Онлайн конвертер координат и яндекс карта.</h3>


## Available Scripts

<p>Для работы и использования требуется установить npm</p>
<p>npm run start - запуск в режиме разработки</p>

<p>npm run build - компиляция из исходных файлов рабочей версии</p>
<p>установка стат. сервера</p>
<p>serve -s build - развертка билда на стат. сервере</p>
